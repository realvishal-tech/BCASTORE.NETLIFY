## 🎨 CUSTOMIZATION OPTIONS

### **Easy Color Change**
Open `classroom.css`:
```css
:root {
  --classroom-primary: #6366f1;      /* Your color */
  --classroom-secondary: #8b5cf6;
  --classroom-accent: #ec4899;
}
```

### **Add More Classes**
Edit `classroom_data.js` and add to arrays:
```javascript
CLASSROOM_DATA.upcomingClasses.push({
  id: 'class-xyz',
  title: 'New Class',
  teacher: 'Prof. Name',
  // ... more fields
});
```

### **Change Subject List**
Edit subjects array in `classroom_data.js`:
```javascript
CLASSROOM_DATA.subjects.push({
  id: 'new-subject',
  name: 'New Subject',
  icon: '🎯'
});
```
