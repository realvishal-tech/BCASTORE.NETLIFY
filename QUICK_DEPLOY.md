## 🛠️ Troubleshooting

### Page loads slowly?
→ Run Lighthouse audit (F12 → Lighthouse)
→ Check Network tab for slow resources

### Dark mode stutters?
→ Clear browser cache (Ctrl+Shift+Delete)
→ Try different browser
→ Report issue with DevTools Performance tab

### CSP error in console?
→ Check what URL is being blocked
→ Update CSP in netlify.toml if needed
→ Use `unsafe-inline` only if necessary

### 404 not showing?
→ Verify netlify.toml has redirects
→ Check 404.html exists
→ Test with invalid URL like `/nonexistent`
