## 🎓 Usage Examples

### Example 1: Mark Attendance (Teacher)
```
1. Open attendance.html
2. Click "✍️ Mark Attendance" tab
3. Select: Class → Teacher → Date
4. Click "Start Marking Attendance"
5. Select status for each student
6. Click "Mark All Present" (optional)
7. Click "Submit Attendance"
8. Done! ✅
```
⏱️ Time: 3 minutes for 30 students

### Example 2: Check Your Attendance (Student)
```
1. Open attendance.html
2. Click "👨‍🎓 My Attendance" tab
3. See your percentage: 75%
4. See your status: ✅ Good Standing
5. Scroll to see recent records
6. Done! ✅
```
⏱️ Time: 1 minute

### Example 3: Generate Report (Admin)
```
1. Open attendance.html
2. Click "📊 View Reports" tab
3. Select: Class (optional)
4. Select: Time Period
5. Click "Generate Report"
6. View statistics
7. Export to CSV (optional)
8. Done! ✅
```
⏱️ Time: 2-3 minutes
